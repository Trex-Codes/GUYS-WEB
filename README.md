# GUYS-WEB
## PROJECT (Artistic Music Group)

## Starting 🚩
_This project was born from a friend's idea to create his own band composed of his friends. It explains how the band was born and its expectations for the future._

## Architecture & Installation 🔧
_The project is built as a **Modern One-Page Application**._
All content is combined into a single `index.html` file with smooth scrolling and CSS animations.

**To run the project:**
Simply open `index.html` in any web browser. No servers or dependencies are required.

## Sections 🎸
_The template is composed of 7 key sections, each explaining an aspect of the band's history and inspiration:_

1. **Home**: Shows the band's motto and identifiable logo.
2. **About Us**: Explains the origin of the band, its evolution, and global expectations.
3. **Discography**: The goal to release a new album that challenges society's thinking, inspired by Gustavo Cerati.
4. **Path**: A glimpse into their first album, expressing their feelings and way of demonstrating themselves to the world.
5. **Record Studio**: A dialogue of how Danex met with his friends to create a classic rock revival band under the name THE GUYS.
6. **Tours and Concerts**: Their dream to perform at legendary venues like Jorge Eliécer Gaitán or Estadio El Campín.
7. **Contacts**: Information about their social networks, next singles, tours, and how to interact with the band.
